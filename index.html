Set Up Your Gemini API Key

Power your email responses with Google's advanced AI technology

1. Access Google AI Studio

Visit Google AI Studio and sign in with your Google account.

2. Create Your API Key

  1. Click "Get API Key" in the top right corner
  2. Select "Create API key"
  3. Copy and securely store your new API key

3. Configure the Extension

  1. Open the AI Email Assistant extension
  2. Click the Settings icon
  3. Paste your Gemini API key
  4. Save your changes
